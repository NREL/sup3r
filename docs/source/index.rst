What is sup3r?
==============
sup3r stands for **Super Resolution for Renewable Resource Data**.

The Super Resolution for Renewable Resource Data software uses
generative adversarial networks to create synthetic high-resolution wind and
solar spatiotemporal data from coarse low-resolution inputs.
