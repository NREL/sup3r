classmethod init_pass_collect(out_dir, file_paths, model_path, fwp_kwargs=None, dc_kwargs=None)[source]

Generate config files for forward pass and collection

Parameters

  • out_dir (str) – Parent directory for pipeline run.

  • file_paths (str | list) – A single source h5 wind file to extract raster data from or a list of netcdf files with identical grid. The string can be a unix-style file path which will be passed through glob.glob

  • model_path (str) – Path to gan model for forward pass

  • fwp_kwargs (dict) – Dictionary of keyword args passed to the ForwardPassStrategy class.

  • dc_kwargs (dict) – Dictionary of keyword args passed to the Collection.collect() method.
